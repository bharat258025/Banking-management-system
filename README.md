💳 C++ Banking System

This project is a simple Banking System built using C++, designed to simulate basic banking operations like account management, transactions, and secure user access.
It demonstrates the use of Object-Oriented Programming (OOP) concepts along with file handling to maintain persistent data.

🚀 Features  

🏦 Account Creation  
Users can create new bank accounts with necessary details.  
💰 Deposit & Withdrawal  
Perform transactions such as depositing and withdrawing money securely.    
📊 Balance Inquiry  
Check the current account balance at any time.    
🧾 Transaction History (Statement)  
View past transactions stored using file handling.  
🔐 User Authentication  
Secure login system with password validation to protect account access.  

🛠️ Technologies Used  

C++  
Object-Oriented Programming (OOP)  
File Handling (for data persistence)  

📂 Project Structure  
Banking-management-System/  
│── accounts.txt

⚙️ How It Works  

The system is built around an Account class that stores user information and handles operations like deposit, withdrawal, and balance checking.
All account data and transaction records are stored in files, ensuring that data remains available even after the program is closed.
Users must log in using their credentials, which adds a layer of security to the system.
